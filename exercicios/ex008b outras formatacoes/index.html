Codigo-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.
Este codigo esta envelopado com a tag CODE, que deixa a fonte monoespaçada, assim cada letra ocupa o mesmo espaço, evitando que algumas letras fiquem ruins de se ver e selecionar.

Abaixo temos um exemplo de codigo em Python:

num = int(input('Digite um numero')) if num % == 0: printf(f'O numero {num} é PAR') else: printf(f'O numero {num} é IMPAR') printf('Fim do Programa')

Abaixo temos o mesmo exemplo de codigo, porem dentro de uma tag PRE, que indenta todas as linhas automaticamente. Tambem é possivel tabular o bloco do codigo inteiro ou algumas partes manualmente posteriormente da maneira desejada:

        
num = int(input('Digite um numero'))
if num % == 0:
    printf(f'O numero {num} é PAR')
else:
    printf(f'O numero {num} é IMPAR')
    printf('Fim do Programa')
