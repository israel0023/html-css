Explicação ID e Class

Seletor personalizado é um seletor onde apenas ele será modificado com uma edição diferente dos demais iguais a ele. Por exemplo o H1 inicial deste site esta com edição diferente. Para isso usamos ID.

Para personalizar um seletor, é necessario identifica-lo dentro da tag dele com um 'id'.

Exemplo: h1 id="principal">Criando Sites com HTML e CSS /h1>

Ja para modificarmos varios seletores com uma mesma edição diferenciada, usamos 'class'.

Exemplo: h2 class="basico">CSS basico /h2>

Informação Importante

Para simplificar a explicação de como fazer um personalização de seletor, podemos fazer a criação como se fosse uma classe de orientaçãoa objetos e depois aplicamos no seletor que queremos personalizar.

ID - Primeiro cria-se o seletor no CSS. Como aqui:

h1#principal { text-align: center; background-color: beige; color: blueviolet; }

Em seguida declaramos a id no seletor que queremos modificar: h1 id="principal">Criando Sites com HTML e CSS /h1>

CLASS - Primeiro cria-se o seletor no CSS. Como aqui:

.basico { color: rgb(64, 188, 23); } .intermediario { color: darkgoldenrod; } .avançado { color: rgb(139, 14, 14); }

Em seguida, declaramos a class nos seletores que queremos modificar: h2 class="basico">CSS basico /h2>

Em HTML é id = em CSS é #

Em HTML é class = em CSS é .

Em HTML digitamos a tag para personalizar o seletor (id e class) e no CSS digitamos um caractere (# e .) seguido do nome escolhido para fazer a correspondencia do seletor.

Observações

Posso colocar duas ou mais classe num mesmo seletor, separando os nomes das class apenas por espaço. Exemplo: h2 class="avançado destaque">CSS Avançado /h2>

Posso mesclar num mesmo seletor id e class, mas o id irá sobrepor as configurações da class na hierarquia. Exemplo: h1 id="principal" class="destaque">Criando Sites com HTML e CSS /h1>
