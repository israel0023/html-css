Explicações

Esta é a forma correta de centralizar um objeto dentro de um site. Irei explicar as tags mais importantes, porem o basico a se saber é que precisamos criar um container, que será uma id com nome container e criar outra id com o conteudo que queremos mover, e usando a tag 'transform: translate(-50%, -50%);' iremos aplicar o movimento do objeto.

Criamos sempre a id container porque ele é que vai ser o que contém o conteudo que queremos mover, por isso o nome container (aquele que comntém algo).

Todo posicionamneto de container é relativo por padrão. Por isso a tag position: relative dentro do id container é somente para expliacção, mas não altera nada no visual de fato.

Já a tag position no conteudo precisa ser absolute, e isso te dá o poder de colocar as tags left e top, colocando o objeto no meio da tela, porem não totalmente centralizado.

Depois disso podemos usar a tag 'transform: translate' para ajustar melhor a posição do objeto, usando os valores como parametro '(-50%, -50%)';

Usando todas estas propriedades, podemos levar o elemento para onde quisermos de forma livre, o que não é possivel fazer sem usar a id container. Isso facilita também para fazermos um elemento se tornar responsivo, como no site do mascote android, onde foi muito dificil fazer isto.
