Usando links externos

Acesse aqui o meu repositorio publico do github

Neste exemplo criamos um link externo com a tag A (A HREF). Após o link do site externo, segue a tag TARGET="_blank", que abre uma nova guia para este link, evitando que o usuario saia da sua pagina. Apos TARGET, temos a tag REL="external", que indica internamente ao nosso site que este é um link externo. Isso serve para mecanismos de busca que vasculham nosso site obter informações e indexar o conteudo do site e dos links externos na busca.


Links de Patrocinio ou Não Totalmente Confiaveis

Este conteudo teste fala sobre a matéria link teste

Aqui a nossa tag REL tem como parametro "nofollow". Este parametro ira indicar internamente que não confiamos totalmente no site do link, ou seja, indicando aos mecanismos de busca para que não indexem este link no conteudo da busca pois não damos aval ao conteudo do site em questão.

Links para download

Aqui foi criada uma lista com 2 linhas, cada uma com um download. A tag A (A HREF) tambem é usada para links de download, porem preciamos usar junto a tag TYPE usando o parametro de 'media type' com a extensão correspondente ao arquivo, o link baixa o arquivo diretamente.

Cada arquivo possui um media type, que pode ser encontrado no site da IANA
