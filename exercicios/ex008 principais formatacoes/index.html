Principais Formatações

Negrito / Destaques

Há dois tipos de formatção de texto em html: Semantico e Não Semantico. O tipo semantico é de uso recomendavel, pois

Nesta frase temos um termo em negrito usando a tag B (não semântica)

Nesta frase temos um termo em destaque usando a tag STRONG (semântica).

Note que negrito e destaque são coisas diferentes, visto que em aparência são a mesma coisa, mas STRONG(destque) por ser uma tag semantica, tem um uso diferenciado e melhor.


Itálico / Ênfase

Nesta frase temos um termo em Itálico usando a tag I (não semantica)

Nesta frase temos um termo em Ênfase usando a tag EM (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK. A cor violeta desta marcação foi inserida usando uma tag CSS global la em cima entre as tags TITLE e HEAD.

E no outro parágrafo, temos outro texto marcado no final. Neste outro texto marcado foi usado uma tag CSS dentro da minha tag MARK dest linha.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo, usando tag BIG para grande (obsoleta, não se usa mais) e tag SMALL para letras pequenas.

Texto deletado (riscado ou atropelado)

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para isso usamos a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase indicando que ele foi adicionado depois, usando a tag INS.

Existe tambem o sublinhado, usando a tag U (não semântica).

Texto sobrescrito

Usado para inserir coisas do tipo x20+3, usando a tag SUP no numero 20.

Texto subescrito

Usado para inserir coisas do tipo H20, usando a tag SUB no numero 2.
