Explicações

Importante: A tag 'margin: auto;' não funciona em elementos inline-level. Por isso adicionamos um 'display: block;', assim a configuração da margem irá funcionar, ja que agora o elemento se torna box-level. Por isso conseguimos colocar o iframe no meio da tela usando a tag 'margin'.

De acordo com o prof Guanabara, neste caso aqui onde estamos utilizando uma tabela como exemplo, seria mais adequado usar uma div com uma tag 'overflow-x ou overflow-y' para mostrar e setar esta tabela. Isso por conta do google ter um problema de indexação na busca por conteudo com iframe. Porem como este exemplo é só pra mostrar como usar uma pagina local dentro do iframe, isso não será problema por aqui. Alem disso, apenas para informação, há uma certa vantagem para configurar o CSS desta forma, pois o iframe permite que configuremos o estilo de forma separada.

Em aulas futuras será explicado mais sobre indexação de busca para paginas com iframe.
