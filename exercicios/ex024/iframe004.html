Explicações

Para garantir a segurança dentro de um iframe é necessario incluirmos algumas tags dentro dele. Vejamos abaixo as tags e o que fazem.

tag SANDBOX: usando o parametro 'sandbox' bloqueia a captura de dados digitados pelo usuario.
Note que há outros parametros tambem usados neste site e eles são colocados todos juntos dentro das aspas, separados apenas por um espaço ficando dessa maneira 'sandbox="allow-same-origin allow-forms allow-scripts". Onde 'allow-same-origin' permite conteudo da mesma origem ou do mesmo servidor, 'allow-forms' permite o uso de formularios e 'allow-scripts' que permite o uso de scripts de javascript. Para melhor entendimento vide codigo do site.

tag REFERRERPOLICY: com o parametro 'no-referrer' desliga a captura de dados para uso em propagandas.
