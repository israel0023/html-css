Explicações

Dentro da tag do iframe, ja aparece a tag 'src' por padrão, onde podemos colocar alguma pagina pra rodar inicialmente. Porem substituindo src pela tag SRCDOC, podemos incluir varios elementos como titulos, paragrafos e ate imagens para serem m ostradas inicialmente no site assim que ele é carregado.

Obs.: Para imagens, temos que substituir as aspas duplas por aspas simples, pois estamos colocando tag com parametro dentro de uma area de parametro, onde ja contem aspas duplas.
