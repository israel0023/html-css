Explicaçõs

Criando sombras

Utilizar a tag 'BOX-SHADOW'. Em seguida ir para o devtools e clicar no botaozinho para visualizar um editor de sombras que te mostra a pré-visualização da sombra e alterar como quiser, em seguida é so copiar a configuração para o codigo.

box-shadow: 7px 6px 6px black;

Arredondamento de caixas

Usando a tag 'BORDER-RADIUS' podemos arredondar as caixas e criar uma certa sofisticação ao estilo. Esta tag funciona como todas que tem 4 lados, onde usando sentido horario (cima, direita, baixo, esquerda) criamos um shorthand que define todos os valores para os lados.

border-radius: 10px 20px 30px 40px;

Criando um circulo

Para criar o circulo perfeito que vemos no final da pagina, Prof Guanabara primeiro criou uma div no final do codigo, dando a ela um ID chamado bola. Em seguida foi no codigo de estilo CSS e setou a div #bola com altura e largura iguais, cor branca e por fim, setou um border-radius de 50%.

Com este border-radius ele tambem mostrou como usar porcentagem como valor para algum elemento ou configuração.

div#bola {
height: 100px;
width: 100px;
margin: 20px;
background-color: white;
border-radius: 50%;
}

div id="bola"> /div
