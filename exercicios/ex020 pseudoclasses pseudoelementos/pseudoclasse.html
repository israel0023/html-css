Pseudoclasses

Pseudoclasses servem para configurar o estado de um elemento. Aqui, no caso, ao elemento DIV.

Explicações

Analisando o codigo fonte vemos que dentro do STYLE, no bloco de codigo seguinte:

div { display: inline-block; background-color: lightgray; border: 1px solid black; text-align: center; height: 200px; width: 200px; }

Temos a edição do elemento (ou seletor) DIV. Ate aqui, nada demais.

Ja neste bloco de codigo:

div:hover { background-color: yellow; }

Temos a configuração da pseudoclasse, no caso, a nossa HOVER (flutuar, pairar). Esta pseudoclasse configura um comportamento quando colocamos o mouse em cima de um determinado elemento, a DIV, neste caso. Assim, quando passamos o mouse o background-color ficara em amarelo.

Observação: Para configurar um i]ID, usamos # (hashtag) na parte do CSS, para configurar uma CLASS, usamos . (ponto), e para configurar uma pseudoclasse usamos : (dois pontos).
