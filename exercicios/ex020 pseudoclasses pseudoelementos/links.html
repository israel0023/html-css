Pseudoelementos

Pseudoelementos são como psudoclasses, mas modificam o conteudo periferico (ao lado) do elemento em questão. Pseudoelementos são representados com dois 'DOIS PONTOS' (::) apos o elemento aser modificado. Exemplo:

a::after { content: '[link]'; text-decoration: none; font-weight: normal; color: darkgray;}

Porem depois modifiquei para:

a::before { content: '✪'; }
a::after { content: '✪'; text-decoration: none; font-weight: normal; }

Depois criei a classe .especial mostrando que anexado a ela tambem pode-se colocar pseudoelementos (::after, ::before). Dessa forma podendo criar uma edição diferente apenas para os links que eu quis destacar nesta pagina tendo simbolos diferentes e um cor de fundo avermelhada.

.especial { background-color: rgb(235, 160, 160); }
.especial::before { content:'☢ ☠' ; }
.especial::after { content:'☠ ☢' ; }

Sempre bom lembrar...

  • # = id
  • . = classe
  • : = pseudoclasse
  • :: = pseudoelemento
  • > = de pai para filho
