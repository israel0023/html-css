Explicações

Analisando o CSS desta pagina temos 3 blocos de codigo que se complementam. Sendo eles:

div > p { display: none; }

div:hover > p{ display: block; color: white; background-color: red; width: 300px; }

div:hover { color: blue; }


Explicanado cada um individualmente temos:

Este bloco de codigo abaixo foi o primeiro bloco que fiz, que apenas cria a HOVER azul no texto 'passe o mouse aqui'.

div:hover { color: blue; }

Em seguida,criei o bloco abaixo e nele, com a tag NONE dentro da tag DISPLAY, configuramos que o P (paragrafo) dentro da DIV não ira aparecer. Pois "div > p" significa que o paragrafo é filho do DIV

div > p { display: none; }

E por ultimo criamos o bloco:

div:hover > p{ display: block; color: white; background-color: red; width: 300px; }

Onde novamente configuramos o P (paragrafo) dentro da DIV e dizemos que ele tambem é filho da HOVER em "div:hover > p".

A tag DISPLAY aqui configura que ele ira aparecer dentro da HOVER 'passe o mouse aqui' e o restante das tags configura a aparencia do paragrafo quando ele aparecer.
