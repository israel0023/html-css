Trabalhando com Listas

Listas Ordenadas

Abaixo temos um exemplo de lista ordenada coma tag OL (ordered list). Este tipo de lista organiza automaticamente quelquer item que voce coloque no meio da lsita. Com a tag TYPE dentro de OL, escolhemos entre os numeradores tipo numero ou outros (1, A, a, I, i). Edentro ainda da mesma tag OL podemos colcar a tag START, que vai dizer em qual posição começara a lista.

  1. Acordar
  2. Tomar Café
  3. Escovar os dentes
  4. Is para faculdade
  5. almoçar
  6. ir para trabalho

Com a tag LI adicionamos elementos a lista. È opcional no HTML5 fechar a tag LI.


Listas Não Ordenadas

Com a tag UL (unordered list), criamos uma lista desordenada, que não possui hierarquia numeral ou do tipo. Aqui na tag TYPE podemos escolher entre circle, disc ou square

  • Pão
  • Leite
  • Café
  • Arroz
  • Manteiga
  • Macarrão

Lista de Definições

Abaixo temos a Lista de Definições com a tag DL. Esta tag com auxilio das tags DT (termo) e DD (descrição), dá ao seu paragrafo um estilo de dicionario.

Estas tag auxiliam o site a performar em buscas do google, pois a busca da preferencia a sites que usam Lista de Definição em seu conteudo.

HTML
Linguagem de marcação para criação do conteúdo de um site
CSS
Linguagem de marcação para dar forma e estilo visual (design) a um site
Javascript
Linguagem de programção para criar a interatividade de um site
