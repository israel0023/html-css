Explicações

Para configurar uma coluna inteira usamos a tag COLGROUP em qualquer lugar do codigo, aqui foi criado no inicio do codigo. Em seguida criamos uma tag COL dentro do grupo para cada coluna que queremos modificar e criamos uma CLASS ou ID para configura-la.

Podemos fazer a configuração de 2 maneiras:

Na primeira forma (Cadastro 1), criamos uma COL para cada coluna e nomeamos cada uma com uma CLASS, depois configuramos as CLASS como queremos.

Já na segunda (Cadastro 2), criamos 2 classes com cores diferentes, depois criamos uma COL para cada coluna setando a class com a cor desejada para cada coluna. Desse jeito criamos o efeito zebrado nas colunas.
