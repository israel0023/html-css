Explicações

Tabelas são muito rígidas e exigem que sejam respeitadas em sua hierarquia. Caso algum dado seja colocado fora da hierarquia d tablea ou em local errado, ele será exibido acima ou abaixo da tabela e não dentro dela.

Podemos escrever as tags 'td' sem o fechamento, mas é recomendavel que sejam fechadas, ate porque o fechamento da tag (/td) teria que ser apagado manualmente, gerando mais trabalho.

Para alinhar o texto dentro das celulas, usamos as tags 'text-align' para alinhamento horizontal: left, center ou right, e para alinhamento do texto na vertical usamos a tag vertical-align com top, middle ou bottom como parametros. No Excel, as celulas com numeros ja tem alinhamento à direita por padrão, porem no HTML isso tem que ser feito manualmente. Aqui foi criada a classe .numero com 'text-align: right'.

Para adicionar celulas avulsas ou linhas e colunas incompletas, simplemente adicionamos celulas a menos dentro da linha.

HIERARQUIA DE TABELAS (simples)

TABLE = tabela

TABLE ROW = linha de tabela

TABLE HEADER = cabeçalho de tabela

TABLE DATA = dado de tabela
