Explicações

Tag CAPTION

Abaixo do primeiro H1 há uma tag chamada 'CAPTION' onde vemos os dizeres 'População das Unidades Federativas'. O efeito desta tag no site fica semelhante a um título, porém o correto a se traduzir de caption é 'legenda'. Colocamos aqui algumas configurações de estilo para que a tag ficasse da forma como está, mas quando ela aparece inicialmente, o texto da tag fica bem pequeno, realmente parecendo apenas uma legenda.

Tag style exclusiva dentro do texto/html

IMPORTANTE: Dentro da tag TH do título 'população' há uma tag style de CSS exclusiva, onde setamos esta coluna para ter 50 pixels. Para isso, setamos a tabela inteira para 400px e depois de setar a tag CSS de largura avulsa nesta coluna, a mesma fica com 50px e o restante da tabela fica com os 350px restantes do tamanho total.

SCOPE

Outro ponto importante é setar cada coluna e cada linha adequadamente com a tag SCOPE, assim configurando semânticamente a tabela e melhorando os site de varias formas.

Efeito zebrado (configuração para linhas separadamente)

Para fazer o efeito zebrado usamos uma pseudo-classe chamada 'NTN-CHILD', com parametro '(odd)', que significa 'ímpar' ou '(even)', que significa 'par'. Ao inves disso podemos usar tambem '2n-1' para fazer o ímpar ou '2n' para fazer o par, porem usar odd ou even é mais simples, mas o 2n é usado para alterar a configuração a cada 2 linhas, 3n para cada 3 linhas, 4n para cada 4 linhas e assim por diante.

Podemos colocar qualquer cor e tambem qualquer configuração, mas quando a tabela ja tem a cor de fundo que queremos, só precisamos setar a cor para as linhas com a segunda cor desejada, aqui neste caso, a cor cinza, pois o branco ja é a cor de fundo.

Cabeçalho fixo de tabela

Importante: Não funciona no Safari. Funciona apenas em navegadores baseados no Chrome, mas com certeza isso vai ser expandido com o passar do tempo.

Primeiramente setamos a posição da tabela com POSITION: RELATIVE;, depois colocamos um seletor para escolher o TH que esta na TR que esta no cabeçalho: THEAD > TR > TH { } e para este seletor setamos as configurções: POSTION: STICKY;, TOP -1px; e BACKGROUND-COLOR: 'mesma cor do cabeçalho'.

Isso fara com que o texto do cabeçalho da tabela grude no topo e não desça junto com a rolagem da tabela, mas é preciso colocar um bakground-color da mesma cor do cabeçalho junto, se não apenas as letras irão ficar fixas e não dará o efeito desejado.
