Estilo Inline

Este metodo de estilo não é utilizado no dia-a-dia de trabalho, sendo, basicamente, uma forma possível de estilo, mas que não serve para ser usado profissionalmente por gerar muito retrabalho e não ser tão funcional.

No estilo inline digitamos as tags direto na linha do conteúdo do site. Exemplo: no paragrafo, no titulo, no subtitulo, etc. Quando seria muito mais pratico e facil colocar todo o estilo CSS numa linha unica. Aqui foram usados os dois exemplos de estilo, tanto o estilo 'inline' quanto o estilo 'interno (ou local)'.
