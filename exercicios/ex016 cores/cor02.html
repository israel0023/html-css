Abaixo eu coloquei a descrição do que foi feito com CSS ao fundo, porem ocorreu um erro onde cada paragrafo escrito esta em uma folha de CSS e não um abaixo do outro na mesma folha, como seria o correto. Espero resolver este erro nas proximas aulas.

O gradiente que vemos ao fundo foi criado dentro da tag STYLE de CSS, dentro do HEAD, logo abaixo de TITLE, que é onde geralmente se coloca as CSS de uma pagina.

O codigo usado foi o seguinte:

STYLE * { height: 100%; } body { background-image: linear-gradient(45deg, #331832, #694D75, #133bdd, #9FC2CC, #F1ECCE); } h1 { color: white; }
